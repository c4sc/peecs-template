## peecs-template

Requerements
---
- ruby
- sass
- bower
- node
- gulp

Installation
---
```
gem install sass
npm install
bower install
gulp
gulp watch  
```
then, access localhost:9000 in your favourite browser

Troubleshooting
---

- make sure that the port 9000 is not used by any other process.
- install livereload extension/addon in browser, and enable it
